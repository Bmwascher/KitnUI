# KitnUI

An ElvUI profile installer for **KitnUI** — guided setup with Dark and Color variants.

KitnUI walks you through configuring ElvUI and popular addons with a step-by-step installer. Choose your visual style, install profiles with a single click, and load them on every alt.

## Features

- **Step-by-step installer** — guided setup with a clean UI
- **Dark & Color variants** — DPS and Healer layouts for both styles
- **One-click profile loading** — apply all installed profiles to alts instantly
- **Per-spec CDM layouts** — Blizzard Cooldown Manager layouts tailored per spec/role
- **Auto-detection** — prompts new characters to load profiles, and notifies when updates are available
- **Smart updates** — only reimport profiles that have changed, skip the rest
- **Per-class CDM** — dedicated `/kitn cdm` flow for Blizzard Cooldown Manager layouts

## Requirements

- **ElvUI** (required)

## Slash Commands

- `/kitn install` — Open the full installer
- `/kitn update` — Reimport only profiles that have been updated
- `/kitn load` — Apply installed profiles to this character
- `/kitn cdm` — Open Blizzard CDM importer for the current class
- `/kitn version` — Show addon and per-profile version info
- `/kitn reset` — Reset installer state

## Related Addons

- **KitnEssentials** — Standalone combat, QoL, and skinning modules
- **KitnUI Lite** — Standalone profile installer for popular addons (no ElvUI required)
